# MarketFix digital business card

Static one-page website for a QR-code business card.

Included:
- Responsive physical-card layout for mobile and desktop.
- Pointer shine on desktop.
- Optional device-orientation shine on supported mobile browsers.
- Minimal contact and company details from the previous website.

GitHub Pages:
Set Pages to deploy from GitHub Actions. The workflow is in .github/workflows/pages.yml.
